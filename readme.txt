== Description ==

Resource light alternate post management screen for use on slow connections and/or slow computers. Publish and edit posts using a basic high contrast HTML editor outside the normal WordPress environment.
Can be used without JavaScript and Flash.
When JavaScript is available it is possible to upload and insert images.
After logging in the browser is redirected directly to the alternate post management screen (this can be turned of with a setting).
One click switches to the standard rich edit screen and back to the alternate post management screen.

Please report any issue you have on the [support page](http://blog.bokhorst.biz/4146/computers-en-internet/wordpress-plugin-light-post/#respond "Marcel's weblog").

See my [other plugins](http://wordpress.org/extend/plugins/profile/m66b "Marcel Bokhorst").

== Installation ==

*Using the WordPress dashboard*

1. Login to your weblog
1. Go to Plugins
1. Select Add New
1. Search for Light Post
1. Select Install
1. Select Install Now
1. Select Activate Plugin

*Manual*

1. Download and unzip the plugin
1. Upload the entire *light-post/* directory to the */wp-content/plugins/* directory
1. Activate the plugin through the Plugins menu in WordPress

== Frequently Asked Questions ==

= How can I change the styling? =

1. Copy *wp-light-post.css* to your upload directory to prevent it from being overwritten by an update
2. Change the style sheet to your wishes
